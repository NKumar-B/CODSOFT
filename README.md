# CodSoft Artificial Intelligence Internship

This repository contains my completed tasks for the Artificial Intelligence internship at [CodSoft](https://www.codsoft.in/). 

## 📋 Tasks Overview

### Task 1: Chatbot with Rule-Based Responses
- **File:** `chatbot.py`
- **Description:** A conversational chatbot built using regular expressions and pattern matching to provide rule-based responses. It is designed to understand basic greetings, inquiries about its identity, and standard farewells.
- **Concepts Applied:** Natural Language Processing (Basics), Pattern Matching, Conversation Flow.
- **How to Run:**
  ```bash
  python chatbot.py
````

### Task 2: Tic-Tac-Toe AI

  - **File:** `tic_tac_toe.py`
  - **Description:** An unbeatable Tic-Tac-Toe AI agent that plays against a human user in the terminal. The AI evaluates all possible future moves to ensure it never loses.
  - **Concepts Applied:** Game Theory, Search Algorithms, Minimax Algorithm.
  - **How to Run:**
    ```bash
    python tic_tac_toe.py
    ```

### Task 3: Image Captioning

  - **File:** `image_captioning.py`
  - **Description:** An AI application that combines computer vision and natural language processing to generate descriptive captions for provided images. It utilizes the powerful BLIP (Bootstrapping Language-Image Pre-training) transformer model.
  - **Concepts Applied:** Computer Vision, Deep Learning, Transformer Models.
  - **How to Run:**
    ```bash
    # Install required libraries first
    pip install transformers torch Pillow

    # Run the script
    python image_captioning.py
    ```

## 🛠️ Technologies & Libraries Used

  - **Language:** Python 3.x
  - **Standard Libraries:** `re`, `math`, `os`
  - **External Libraries:** `transformers`, `torch`, `Pillow`
  - **Pre-trained Models:** `Salesforce/blip-image-captioning-base` (via Hugging Face)
