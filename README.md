# CodSoft Artificial Intelligence Internship

Welcome to my repository showcasing the projects I completed during the **Artificial Intelligence Internship at CodSoft**. This collection highlights my practical understanding of AI concepts, problem-solving skills, and hands-on experience with real-world applications.

---

## About the Internship

This internship provided me with an opportunity to explore core areas of Artificial Intelligence, including:

* Natural Language Processing (NLP)
* Game AI & Decision Making
* Computer Vision & Deep Learning

Each task was designed to strengthen both theoretical knowledge and practical implementation skills using Python.

---

## Tasks Overview

### Task 1: Chatbot with Rule-Based Responses

* **File:** `chatbot.py`

* **Description:**
  A simple conversational chatbot built using **regular expressions and pattern matching**. It can understand and respond to:

  * Greetings (Hi, Hello, etc.)
  * Basic identity-related questions
  * Farewell messages

* **Key Concepts:**

  * Natural Language Processing (Basics)
  * Pattern Matching
  * Rule-Based Systems

* **How to Run:**

```bash
python chatbot.py
```

---

### Task 2: Tic-Tac-Toe AI

* **File:** `tic_tac_toe.py`

* **Description:**
  An **unbeatable Tic-Tac-Toe AI agent** that plays against a human in the terminal. The AI uses advanced decision-making to evaluate all possible moves and ensure it never loses.

* **Key Concepts:**

  * Game Theory
  * Minimax Algorithm
  * Search Algorithms

* **How to Run:**

```bash
python tic_tac_toe.py
```

---

### Task 3: Image Captioning

* **File:** `image_captioning.py`

* **Description:**
  An AI-powered application that generates descriptive captions for images using the **BLIP (Bootstrapping Language-Image Pre-training)** model.

* **Key Concepts:**

  * Computer Vision
  * Deep Learning
  * Transformer Models

* **Setup & Run:**

```bash
# Install required libraries
pip install transformers torch Pillow

# Run the script
python image_captioning.py
```

---

## Technologies & Tools used in Task3

* **Programming Language:** Python 3.x
* **Core Libraries:**

  * `re` (Regular Expressions)

  * `transformers`
